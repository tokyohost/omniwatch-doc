# 第 1 课：做出第一块屏幕

[上一课](00-准备工作.md) · [返回总目录](../README.md) · [下一课](02-读取监控数据.md)

这一课的目标很朴素：黑色背景上显示标题和一句欢迎文字。先把闭环跑通，再谈炫技。

## 第一步：复制完整示例

下载或复制 [style_hello.py](code/style_hello.py)，不要只复制文档中的零散片段。文件结构如下：

```python
from config import BLACK, BLUE, HEIGHT, WHITE, WIDTH
from styles.style_plugins import register_style


class HelloStyle:
    """绘制入门欢迎界面。"""

    name = "hello"
    zh_name = "你好世界"
    type = "custom"
    width = WIDTH
    height = HEIGHT
    landscape = False
    font_name = "native"
```

三项元数据的作用：

- `name`：程序使用的唯一标识，只能写小写字母、数字、下划线。
- `zh_name`：界面展示给人的中文名称，可以自由修改。
- `type`：自定义样式固定写成 `"custom"`。

`width`、`height` 和 `landscape` 描述屏幕方向。本系列先保持竖屏。

## 第二步：理解三个必需方法

### `create_dirty_regions`

它告诉渲染器哪些区域以后可以单独刷新。第一版没有动态内容，先返回空列表：

```python
@staticmethod
def create_dirty_regions():
    """声明本样式没有需要单独更新的动态区域。"""
    return []
```

### `draw_visible`

完整显示或首次显示时，渲染器会分条带调用它。先清背景，再画内容：

```python
def draw_visible(self, canvas, snapshot):
    """绘制当前条带中可见的固定内容。"""
    canvas.clear(BLACK)
    canvas.fill_rect(8, 8, 224, 28, BLUE)
    canvas.text(16, 18, "OMNIWATCH", WHITE)
    canvas.text(16, 64, "HELLO!", WHITE, 2)
```

Canvas 自动裁剪，所以这几个命令即使每个条带都执行也不会画到错误位置。

### `draw_dirty`

局部区域变化时调用。当前没有动态区域，保留一个空实现：

```python
def draw_dirty(self, canvas, key, snapshot):
    """处理动态区域重绘；入门样式暂时没有动态区域。"""
    return None
```

参数顺序必须是 `canvas, key, snapshot`。

## 第三步：注册样式

文件末尾需要一个工厂函数和一次注册：

```python
def create_hello_style():
    """创建你好世界样式实例。"""
    return HelloStyle()


register_style(HelloStyle.name, create_hello_style)
```

注册缺失、重复注册或注册了另一个名称，上传校验都会失败。
